# Welcome to Mini SSG

This is a minimal static site generator built with Python.

## Features

- Markdown to HTML conversion
- Simple template system
- Local development server

## Getting Started

1. Write your content in markdown files
2. Run `mini-ssg build ./content ./output`
3. Preview with `mini-ssg serve ./output`

print("Hello, World!")
